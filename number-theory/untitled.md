# Division and Greatest common divisor

### Division

Let $$\mathbb{Z} = \{\dots , -1, 0, 1, 2, 3 \dots \}$$be the set denoting the integers

**Definition - Division**

> For $$a, b, \in \mathbb{Z} $$we say that $$a$$divides $$b$$if there is some $$k \in \mathbb{Z}$$such that $$a \cdot k = b$$
>
> _Notation:_ $$a | b$$
